Cost Minimization

Theory of Production — Lesson Summary & Practice Quiz

01

Defining Economic Costs

Accounting Cost

Actual out-of-pocket expenses: historical costs, depreciation, and bookkeeping entries. Used for financial reporting.

Economic Cost

Cost of utilizing resources in production, including opportunity cost. Always ≥ accounting cost.

Opportunity Cost

Cost of the forgone best alternative use of a resource. Central to economic reasoning.

Labor Cost (Economic View)

Valued at the wage rate (w) — what labor could earn in its best alternative employment (opportunity cost).

Capital Cost (Economic View)

Not historical purchase price (sunk cost), but the rental rate (r) — what someone else would pay to use that capital.

Rule: Cost of input use = Price per unit of input × Amount of input used. A firm's total cost is C = wL + rK.
02

Fixed vs. Variable Costs

Total Cost (TC)

Full economic cost of production. TC = TFC + TVC

Fixed Cost (TFC)

Does not vary with output. Cannot be eliminated except by shutting down entirely.

Variable Cost (TVC)

Changes as output changes. Includes labor, raw materials, and other inputs that can be adjusted.

03

The Isocost Line

An isocost line shows all combinations of K and L that can be purchased for a given total expenditure C.

Cost Equation C = wL + rK Rewritten for K (linear form) K = C/r − (w/r) · L Key parameters K-intercept = C/r  |  L-intercept = C/w  |  Slope = −w/r
🔄 Isocost Shifts (C changes)

If C decreases, the isocost shifts inward (parallel shift), keeping slope −w/r unchanged. A higher C shifts it outward.

🔃 Isocost Rotates (w or r changes)

If w increases, the L-intercept (C/w) decreases → the isocost rotates inward along the L-axis. The slope −w/r steepens.

04

Cost-Minimizing Input Choice

To minimize cost for a fixed output Q₀, the firm finds the point where the isoquant is tangent to the lowest possible isocost.

Tangency Condition (slope of isoquant = slope of isocost) MRTSLK = w / r Equivalently (via MRTS = MP_L / MP_K) MP_L / MP_K = w / r Rearranged → Equi-Marginal Condition MP_L / w = MP_K / r
Equi-marginal principle: At the optimum, the last dollar spent on labor yields the same marginal output as the last dollar spent on capital. If MP_L/w > MP_K/r, the firm should hire more labor (and less capital) to reduce cost.
05

Lagrange Multiplier Method

1
Objective: Min C = wL + rK
2
Constraint: Q₀ = f(K, L) — must hit the target output
3
Lagrangian: Z = wL + rK − λ[Q₀ − f(K,L)]
4
FOCs: ∂Z/∂L = 0 → w = λ·MP_L  |  ∂Z/∂K = 0 → r = λ·MP_K  |  ∂Z/∂λ = 0 → Q₀ = f(K,L)
5
Divide FOC(i) by FOC(ii): This yields w/r = MP_L/MP_K → the tangency condition. Use this ratio equation plus the constraint to solve for K* and L*.
📐 Worked Example
Given: Q = 10·K0.4·L0.6,  w = 50,  r = 100,  Q₀ = 1000
FOC ratio → K/L = 1/2 → K = L/3 (or L = 3K)
Sub into constraint: 1000 = 10·K0.4·(3K)0.6
Solve → K* ≈ 51.73,  L* ≈ 155.18

06

Input Price Changes & the Expansion Path

When Input Price Changes

If w increases (holding Q₀ constant), the isocost rotates. The new tangency point moves to less labor (L₂ < L₁) and more capital (K₂ > K₁) — the firm substitutes away from the now-pricier input.

Expansion Path

If input prices are fixed and desired output varies, the locus of all cost-minimizing (K*, L*) combinations traces the firm's expansion path. Each point on it is a tangency between an isocost and an isoquant.

Key takeaway: The expansion path connects the cost-minimizing input bundles as Q rises — it shows how input use scales with output, holding relative prices constant.
